# EnchantInnovation

EnchantInnovation introduces item experience and levels to Minecraft 1.21.5. The mod works across multiple platforms and contains module folders `common`, `neoforge` and `fabric`.

## Building

1. Install **JDK 21**.
2. Run `./gradlew build` in the repository root.

## Features

* Items gain experience and levels when used.
* NeoForge events award XP for actions like breaking blocks, hitting entities with arrows or blocking with a shield.
